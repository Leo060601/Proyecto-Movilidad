
# Proyecto Movilidad

## Estructura del Proyecto

- `backend/`: Servidor Node.js/Express para la API y base de datos.
- `frontend/`: Aplicación React para la visualización y gestión de rutas.

## Cómo ejecutar el proyecto

1. Instala dependencias en cada carpeta:
   - `cd backend && npm install`
   - `cd frontend && npm install`
2. En desarrollo, abre dos terminales:
   - Backend: `cd backend && npm run dev`
   - Frontend: `cd frontend && npm start`
3. El backend estará en http://localhost:5000 y el frontend en http://localhost:3000

## Organización

Cada carpeta tiene su propio README con detalles de estructura, dependencias y funcionalidades.
